## 开发
开发过程仅对核心功能的实现方式进行记录。

### 特征参数
每个用户都具备一个特征参数，特征参数将会被转译为query参数进行数据库查询，特征参数描述了一个用户的兴趣喜好。为了能够使匹配更加精准，则需要一个默认的基准线数据。

``` json
{
    /* 角色 fixed */
    'role': [
        'S': 1.0,
        'Dom': 1.0,
        'M': 0.0,
        'Sub': 1.0,
        'Switch': 0.0,
    ],
    /* 偏好 mutable */
    'into': [
        'hair': 0.8,
        'stocking': 0.6,
        'heels': 0,
    ]
}
```
**特征用户算法** 待定

**偏好词和偏好词库** 预设词+用户数据流（帖子，动态，聊天，评论）中提取出来的高频词,筛选功能中可以进行筛选
